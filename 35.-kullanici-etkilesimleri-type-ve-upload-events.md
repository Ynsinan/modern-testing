# 🎤 35. Kullanıcı Etkileşimleri - Type ve Upload Events

**Type (Yazma):** RTL'de bir metin girişi veya metin alanındaki metin yazma işlemi, userEvent.type() fonksiyonu kullanılarak gerçekleştirilir. Bu işlev, belirli bir metni bir metin alanına veya giriş kutusuna yazarken kullanılır.

**Upload (Yükleme):** RTL'de bir dosyanın yüklenmesi işlemi, userEvent.upload() fonksiyonu ile gerçekleştirilir. Bu işlev, bir dosyanın seçilip yüklendiği bir dosya yükleme alanı veya düğme bileşeni için kullanılır.

Şimdi verilen kod bloğunu yorum satırlarıyla açıklayalım:

{% code title="App.test.js" lineNumbers="true" %}
```javascript
import { render, screen } from '@testing-library/react';
import userEvent from '@testing-library/user-event';

function MyComponent() {
  return (
    <div>
      <select multiple>
        <input />
      </select>
    </div>
  );
}

it('should be type', async () => {
  // Kullanıcı etkileşimlerini simüle etmek için userEvent kütüphanesini kullanıyoruz.
  const user = userEvent.setup();
  render(<MyComponent />); // MyComponent bileşenini ekrana render ediyoruz.

  const inputElement = screen.getByRole('textbox'); // "inputElement" adlı metin alanını buluyoruz.

  // "Hello, World!" metnini metin alanına yazıyoruz.
  await user.type(inputElement, 'Hello, World!');
  // Metin alanının değerini kontrol ederek yazılan metni doğruluyoruz.
  expect(inputElement).toHaveValue('Hello, World!');
});

```

{% code title="App.test.js" lineNumbers="true" %}
```javascript
import React from 'react';
import { render, screen } from '@testing-library/react';
import userEvent from '@testing-library/user-event';

function FileUploadComponent() {
  const handleFileChange = (e) => {
    // Dosya yükleme işlemi burada işlenir.
    console.log('Dosya yüklendi:', e.target.files[0].name);
  }

  return (
    <div>
      <input
        type="file"
        id="fileInput"
        onChange={handleFileChange}
      />
    </div>
  );
}

it('should upload a file', async () => {
  const user = userEvent.setup();
  render(<FileUploadComponent />);

  const fileInput = screen.getByLabelText('Dosya Yükle'); // Dosya yükleme alanını etiketle buluyoruz.

  // Bir dosya yüklemek için kullanılacak sanal bir dosya oluşturuyoruz.
  const file = new File(['dummy file content'], 'example.txt', { type: 'text/plain' });

  // Dosyayı yükleme alanına yüklüyoruz.
  await user.upload(fileInput, file);

  // Dosyanın yüklendiğini doğruluyoruz. Bu, dosyanın işlenip konsola loglandığını kontrol etmek için kullanılabilir.
  // Bu işlem uygulamanın işlevselliğine bağlıdır.
  // Örneğin, dosya adını göstermek için bir ekranın içeriğini kontrol edebilirsiniz.
});

```
