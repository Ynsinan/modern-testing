# 🖥️ 17. Queries - getByDisplayValue

Bu işlev, bir giriş alanının mevcut değerini kullanarak, bu değere sahip giriş alanını seçmenizi sağlar. Bu işlev, özellikle formların mevcut değerlerini doğrulamak veya değiştirmek için kullanılır.

Aşağıda `getByDisplayValue` işlemini kullanarak bir örnek gösterilmektedir:

```javascript
import { render, screen } from "@testing-library/react";

const TestComponent = () => (
	<div>
		<input type="text" value="We Writing Test" />
		<button>Update</button>
	</div>
);

it('should find the input field by its display value', () => {
	render(<TestComponent />);
	const inputField = screen.getByDisplayValue("We Writing Test");
	expect(inputField).toBeInTheDocument();
});

it('should find the button by its text', () => {
	render(<TestComponent />);
	const button = screen.getByText("Update");
	expect(button).toBeInTheDocument();
});

```

Yukarıdaki örnekte, `getByDisplayValue` işlevi "John Doe" değerine sahip olan giriş alanını seçer. Bu işlev, bir giriş alanının mevcut değerini doğrulamak veya bu değeri içeren öğeyi seçmek için kullanışlıdır. Özellikle form alanlarının doğru bir şekilde doldurulduğunu veya belirli bir değere sahip olduğunu test etmek için kullanılır.

`getByDisplayValue` işlevi, form işlemleri ile ilgili testler yaparken ve form öğelerinin mevcut değerlerini doğrularken sıkça kullanılır. Bu sayede, kullanıcıların form alanlarına doğru bilgileri girdiğini veya mevcut bilgileri doğru bir şekilde görüntülediğini doğrulayabilirsiniz.

Özetle, `getByDisplayValue` işlevi, bir giriş alanının mevcut değerine göre seçim yapmanıza ve test etmenize olanak tanır.
