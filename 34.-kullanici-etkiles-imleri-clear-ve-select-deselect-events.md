# 🪘 34. Kullanıcı Etkileşimleri - Clear ve Select/Deselect Events

Bu bölümde, RTL ile clear, select ve deselect işlemlerini ele alacağız.

**Clear (Temizleme):** RTL ile bir form elemanındaki mevcut değerleri temizlemek için userEvent.clear() fonksiyonu kullanılır. Bu işlem, bir girdi alanındaki metin veya bir seçenek kutusundaki seçilen öğeleri temizlemek için kullanılabilir.

**Select (Seçme):** RTL ile bir seçenek kutusundan veya açılır menüden bir veya birden fazla öğeyi seçmek için userEvent.selectOptions() fonksiyonu kullanılır. Bu fonksiyon, bir seçenek kutusundaki veya açılır menüdeki belirli öğeleri seçmek için kullanılır.

**Deselect (Seçimi Kaldırma):** RTL ile seçenek kutusundan veya açılır menüden bir veya birden fazla öğeyi seçimi kaldırmak için userEvent.deselectOptions() fonksiyonu kullanılır. Bu fonksiyon, seçenek kutusundan veya açılır menüden belirli öğelerin seçimini kaldırmak için kullanılır.

Şimdi, verdiğiniz kod bloğunu yorum satırlarıyla açıklayalım:

{% code title="App.test.js" lineNumbers="true" %}
```javascript
import { render, screen } from '@testing-library/react';
import userEvent from '@testing-library/user-event';

function MyComponent() {
  return (
    <div>
      <select multiple>
        <option value="elma">Elma</option>
        <option value="armut">Armut</option>
        <option value="muz">Muz</option>
      </select>
    </div>
  );
}

it('should be selected', async () => {
  // Kullanıcı etkileşimlerini simüle etmek için userEvent kütüphanesini kullanıyoruz.
  const user = userEvent.setup();
  render(<MyComponent />); // MyComponent bileşenini ekrana render ediyoruz.

  const fruits = screen.getByRole('listbox'); // "fruits" adlı seçenek kutusunu buluyoruz.

  // "Elma" seçeneğini seçiyoruz.
  await user.selectOptions(fruits, ['Elma']);
  // "Elma" seçeneğinin seçildiğini kontrol ediyoruz.
  expect(screen.getByRole('option', { name: 'Elma' }).selected).toBe(true);

  // "Elma" seçeneğinin seçimini kaldırıyoruz.
  await user.deselectOptions(fruits, ['Elma']);
  // "Elma" seçeneğinin seçilmediğini kontrol ediyoruz.
  expect(screen.getByRole('option', { name: 'Elma' }).selected).toBe(false);
});

```
