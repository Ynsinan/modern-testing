# 🧩 31. Kullanıcı Etkileşimleri - Mouse/Pointer Events

Eventlere girmeden önce setup kavramını öğrenmeliyiz.&#x20;

`userEvent` ile `setup()` kullanımı Jest testlerinde kullanıcı etkileşimlerini daha temiz ve tekrar kullanılabilir hale getirmek için yapılan bir işlemdir. `setup()` işlevi, kullanıcı etkileşimlerini testlerinizin başında tanımlamanıza ve bu etkileşimleri daha sonra kullanmak üzere saklamanıza olanak tanır. Bu, testlerinizi daha okunaklı hale getirir ve tekrar kullanılabilirliği artırır.

Şimdi bir count componenti yazalım ve setup'ı kullanalım.&#x20;

{% code title="App.test.js" lineNumbers="true" %}
```javascript
import { render, screen } from '@testing-library/react';
import userEvent from '@testing-library/user-event';
import { useState } from 'react';

function MyComponent() {
  const [count,setCount] = useState(0);

  const handleIncrement = () => {
    setCount(count + 1);
  };

  return (
    <div>
      <h1>{count}</h1>
      <button onClick={handleIncrement}>Increment</button>
    </div>
  );
}

it('should be increase', async () => {
  const user = userEvent.setup();
  render(<MyComponent />);
});
```

İlk başta mouse eventlerini göreceğiz. Yani mouse ile herhangi bir işlemi taklit etmek. Bunun için [pointer](https://testing-library.com/docs/user-event/pointer/) kullanacağız.&#x20;

{% code title="App.test.js" lineNumbers="true" %}
```javascript
import { render, screen } from '@testing-library/react';
import userEvent from '@testing-library/user-event';
import { useState } from 'react';

function MyComponent() {
  const [count,setCount] = useState(0);

  const handleIncrement = () => {
    setCount(count + 1);
  };

  return (
    <div>
      <h1>{count}</h1>
      <button onClick={handleIncrement}>Increment</button>
    </div>
  );
}

it('should be increase', async () => {
  // Kullanıcı etkileşimlerini simüle etmek için userEvent.setup() kullanılır.
  const user = userEvent.setup();

  // MyComponent bileşeni ekrana render edilir.
  render(<MyComponent />);

  // Kullanıcı, "Increment" adlı düğmeye tıklama eylemini simüle eder.
  await user.pointer({
    keys: '[MouseLeft]', // Fare sol düğmesini tıklar.
    target: screen.getByRole('button', { name: /increment/i }), // "Increment" düğmesini hedefler.
  });

  // "screen.findByRole()" ile belirli bir role sahip öğe beklenir, bu durumda bir başlık öğesi.
  const headingElement = await screen.findByRole('heading');

  // Beklenen sonucu test eder, "1" başlığının ekranda görüntülenip görüntülenmediğini kontrol eder.
  expect(headingElement).toHaveTextContent('1');
});
```
{% endcode %}

Neden findBy kullandık dersek. Domun update olduğundan emin olduktan sonra expect ettik.&#x20;
