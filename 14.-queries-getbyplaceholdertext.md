# 🏷️ 14. Queries -getByPlaceholderText

Bu işlev, bir HTML öğesinin etiketini (label) kullanarak, bu etiketi ilişkilendirdiği form veya kontrol öğesini seçmenizi sağlar. Bu özellikle erişilebilirlik testleri yaparken ve for/label elementlerini kullanarak form öğelerini etiketlemişseniz çok kullanışlıdır.

Aşağıda `getByLabelText` işlevini kullanarak nasıl bir test yapabileceğinizi gösteren bir örnek bulunmaktadır:

```javascript
import { render, screen } from "@testing-library/react";

const TestComponent = () => (
	<div>
		<label htmlFor="inputField">Input Field:</label>
		<input id="inputField" type="text" />
		<button>Submit</button>
	</div>
);

it('should find the input field by its label', () => {
	render(<TestComponent />);
	const inputField = screen.getByLabelText("Input Field:");
	expect(inputField).toBeInTheDocument();
});
```

Yukarıdaki örnek, `getByLabelText` işlevini kullanarak "Input Field" etiketi ile ilişkilendirilmiş olan input alanını seçer. Bu işlem, sayfanızda daha karmaşık form öğelerini test etmek ve bu öğeleri etiketlerine göre bulmak için çok kullanışlıdır.

Özetle, `getByLabelText` işlevi, form öğelerini etiketlerine göre bulmak için kullanılır ve bu, erişilebilirlik testleri yaparken ve kullanıcı arayüzünü test ederken önemlidir.
