# 🎸 28. Queries - DOM Query Selector

Şimdiye kadar ki gördüğümüz queryler ile hiçbir elementi seçemiyorsak. DOM query selectorler ile bu işlemi yapabiliriz. Bu zamana kadar test yazarken `render` fonksiyonunu kullandık. Aslında render bize birden fazla şey return ediyor. Nedir peki bunlar&#x20;

<figure><img src=".gitbook/assets/image (2).png" alt=""><figcaption><p>Önerilenler.</p></figcaption></figure>

Destruct ederek bunları detaylıca inceleyebiliriz. Bunlardan birisi `container`. Biz bunu kullanacağız. `container` nesnesi, render edilen bileşenin tüm DOM yapısını temsil eder.

{% code title="App.test.js" lineNumbers="true" %}
```javascript
import { render, screen, waitForElementToBeRemoved } from '@testing-library/react';

function MyComponent() {

  return (
    <div>
      <p className='primary-text'>Modern Testing</p>
    </div>
  );
}

it('query selector', async () => {
  const { container } = render(<MyComponent />);
  const element = container.querySelector('.primary-text');
  expect(element.textContent).toBe('Modern Testing');
});
```
{% endcode %}

Sadece className değil herhangi bir data attribute ile de select işlemi yapabiliriz.&#x20;
