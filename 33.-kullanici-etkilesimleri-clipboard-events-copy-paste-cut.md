# 🎲 33. Kullanıcı Etkileşimleri - Clipboard Events - Copy, Paste, Cut

React Testing Library ile kullanıcı etkileşimlerini test etmek, uygulamanızın gerçek dünya kullanıcıları tarafından nasıl kullanılacağını simüle etmek için önemlidir. Clipboard olayları (kopyalama, yapıştırma, kesme) da bu etkileşimlerden biridir. Kullanıcıların metin veya verileri panoya kopyalama, panodan yapıştırma veya bir metin seçerek kesme işlemlerini test etmek için @testing-library/user-event ve fireEvent gibi kütüphaneleri kullanabilirsiniz.

{% code title="App.test.js" lineNumbers="true" %}
```javascript
import { render, screen } from '@testing-library/react';
import userEvent from '@testing-library/user-event';

function MyComponent() {

  const handleCopy = () => {
    // Panoya "Hello World" metnini yazma işlemini simüle ederiz.
    window.navigator.clipboard.writeText("Hello World");
  };

  return (
    <div>
      <button onClick={handleCopy}>Copy</button>
      <input type='text' />
    </div>
  );
}

it('should be copy', async () => {
  // Kullanıcı etkileşimlerini simüle etmek için userEvent.setup() kullanılır.
  const user = userEvent.setup();
  render(<MyComponent />);
  
  // Klavye etkileşimleri:

  // [Tab] tuşuna basma işlemini simüle ederiz.
  await user.keyboard("[tab]");
  // [Enter] tuşuna basma işlemini simüle ederiz.
  await user.keyboard("[enter]");

  // Input öğesini bulur ve odaklanır.
  const inputElement = screen.getByRole('textbox');
  inputElement.focus();

  // Panodan yapıştırma işlemini simüle ederiz.
  await user.paste();

  // Beklenen sonucu test ederiz: Input öğesinin "Hello World" değerine sahip olması gerekir.
  expect(inputElement).toHaveValue('Hello World');
});

```
