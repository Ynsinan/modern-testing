# 🥴 18. Queries - getByAltText

Bu işlev, bir HTML görüntüsünün veya `<img>` öğesinin `alt` özniteliği (attribute) ile eşleşen bir alternatif metni kullanarak bu görüntüyü veya öğeyi seçmenizi sağlar. Bu işlev, sayfanızdaki görsellerin alternatif metinlerini doğrulamak ve test etmek için kullanışlıdır. Aynı zamanda web sitelerinin erişilebilirliğini değerlendirmek için de önemlidir.

İşte `getByAltText` işlevini kullanarak bir örnek:

```javascript
import { render, screen } from "@testing-library/react";

const TestComponent = () => (
	<div>
		<img src="#" alt="Test Fotosu" />
	</div>
);

it('should find the image by its alt text', () => {
	render(<TestComponent />);
	const image = screen.getByAltText("Test Fotosu");
	expect(image).toBeInTheDocument();
});

it('should find the image src', () => {
	render(<TestComponent />);
	const image = screen.getByAltText("Test Fotosu");
	expect(image.src).toBe("http://localhost/#");
	// Burada localhost yerine sadece "#" koysaydık test fail olurdu.
});

```

ukarıdaki örnekte, `getByAltText` işlevi "A beautiful landscape" alternatif metnini içeren `<img>` öğesini seçer. Bu işlev, özellikle resimlerin doğru alternatif metinlere sahip olduğunu doğrulamak için kullanışlıdır. Ayrıca, erişilebilirlik standartlarını karşıladığınızdan emin olmak için de kullanabilirsiniz.

`getByAltText` işlevi, görüntülerin alternatif metinlerini test etmek ve bu görüntülerin sayfa üzerinde doğru bir şekilde yer aldığını doğrulamak için kullanılır. Bu, kullanıcı deneyiminin geliştirilmesi ve erişilebilirliğin sağlanması açısından önemlidir.

Özetle, `getByAltText` işlevi, bir görüntünün `alt` özniteliği ile eşleşen alternatif metni kullanarak bu görüntüyü seçmenizi sağlar ve test etmenize olanak tanır.
