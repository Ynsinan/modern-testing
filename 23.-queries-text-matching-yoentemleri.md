# 🍡 23. Queries - Text Matching Yöntemleri

1.  **Metin İçeriği Eşleştirme (Exact Text Match)**: Bu yöntem, tam olarak belirtilen metni içeren ilk öğeyi seçer. Eğer metin yalnızca bir kez görünüyorsa ve bu metni test etmek istiyorsanız kullanışlıdır.

    ```javascript
    javascriptCopy codeconst element = screen.getByText("Hello, World!");
    expect(element).toBeInTheDocument();
    ```
2.  **Metin İçeriği Eşleştirme (Exact Text Match) ve Diğer Özelliklerle Filtreleme**: Bu yöntem, belirtilen metni içeren öğeleri, diğer seçeneklerle filtrelemek için kullanılır. Örneğin, belirli bir rol ile etiketlenmiş öğeleri veya bir üst öğeyi belirtmek için kullanabilirsiniz.

    ```javascript
    const element = screen.getByText("Hello, World!", { selectorOptions });
    expect(element).toBeInTheDocument();
    ```
3.  **Metin İçeriği Eşleştirme (Exact Text Match) ve Rol ile Filtreleme**: Bu yöntem, belirtilen rol ile eşleşen öğeleri ve bu öğelerin içeriğinde belirtilen metni içeren öğeyi seçer. Özellikle belirli bir rol ile etiketlenmiş öğeleri test etmek için kullanışlıdır.

    ```javascript
    const button = screen.getByRole("button", { name: "Submit" });
    expect(button).toBeInTheDocument();
    ```
4.  **Metin İçeriği Eşleştirme (Exact Text Match) ve Diğer Metinlerle Eşleştirme**: Bu yöntem, belirli bir düzenli ifadeyi eşleştiren metinleri seçmek için kullanılır. Bu, metinlerin belirli bir deseni izlediği durumlar için uygundur.

    ```javascript
    const element = screen.getByText(/Hello/i);
    expect(element).toBeInTheDocument();
    ```
5.  **Metin İçeriği Eşleştirme (Exact Text Match) ve Alternatif Metin Eşleştirme**: Bu yöntem, belirtilen metni içeren, ancak tam olarak eşleşmesi gerekmez. Başka metinlerle kısmi eşleşme yapacak öğeleri seçmek için kullanılır.

    ```javascript
    const element = screen.getByText("Hello", { exact: false });
    expect(element).toBeInTheDocument();
    ```
6.  **Metin İçeriği Eşleştirme (Exact Text Match) ve Metin Dizini ile Eşleştirme**: Bu yöntem, bir işlevi kullanarak özelleştirilmiş bir metin filtresi uygular. Bu, daha karmaşık metin eşleştirme senaryoları için kullanışlıdır.

    ```javascript
    const element = screen.getByText((content, element) => content.includes("Hello"));
    expect(element).toBeInTheDocument();
    ```

Bu açıklamalar ve örnekler, metin eşleştirme yöntemlerini daha iyi anlamanıza yardımcı olmalıdır. Bu yöntemleri kullanarak, testlerinizi daha spesifik hale getirebilir ve kullanıcı arayüzünüzü doğrulayabilirsiniz.
