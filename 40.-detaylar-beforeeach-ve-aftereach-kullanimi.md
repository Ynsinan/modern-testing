# 🐊 40. Detaylar - beforeEach ve afterEach Kullanımı

Mantığı çok basit test yazarken aynı şeyleri defalarca yazmamız gerekiyor. Örnek vermek gerekirse her bir testte aynı componenti render etmek kod tekrarına sokuyor. Temel mantık her bir test çalışmadan önce veya çalıştıktan sonra çalışıyorlar.

`beforeEach` ve `afterEach` özellikle tekrar eden hazırlık ve temizleme işlemleri için kullanışlıdır. Örneğin, her test öncesi belirli bir bileşeni render etmek veya her test sonrasında bir kaynak bağlantısını kapatmak gibi işlemleri gerçekleştirebilirsiniz.

İşte beforeEach ve afterEach kullanımına dair bir örnek:

```javascript
import { render, screen } from '@testing-library/react';
import MyComponent from './MyComponent';

// beforeEach: Her test öncesi çalışacak işlemler
beforeEach(() => {
  // Örneğin, her test öncesi bir bileşeni render etmek
  render(<MyComponent />);
});

// afterEach: Her test sonrası çalışacak işlemler
afterEach(() => {
  // Örneğin, her test sonrası ekranı temizlemek
  screen.cleanup();
});

// Test 1
it('should render MyComponent with initial state', () => {
  // İlk render öncesi durum kontrolü
  expect(screen.getByText(/initial state/i)).toBeInTheDocument();
});

// Test 2
it('should render MyComponent with updated state', () => {
  // İlk render öncesi durum kontrolü
  expect(screen.getByText(/initial state/i)).toBeInTheDocument();

  // Önceki testten gelen etkileri temizlemek için cleanup işlemi
  screen.cleanup();

  // Yeni bir bileşeni render etmek
  render(<MyComponent updatedState={true} />);

  // Güncellenmiş durumu kontrol etmek
  expect(screen.getByText(/updated state/i)).toBeInTheDocument();
});

```

Yukarıdaki örnekte, beforeEach bloğu, her test öncesi MyComponent bileşenini render etmek için kullanılır. afterEach bloğu ise her test sonrasında ekranı temizlemek için kullanılır. Temizleme işlemi, önceki testlerden gelen etkileri ortadan kaldırmak ve bir sonraki testi temiz bir durumda başlatmak için kullanılır.

Bu yöntem, testlerinizin düzenli ve bağımsız olmasına yardımcı olabilir. Ancak, her test sonrasında yapılan temizlik işlemlerine dikkat etmek önemlidir, çünkü bazen beklenmeyen davranışlara neden olabilirler.
