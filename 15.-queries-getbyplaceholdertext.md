# 🗿 15. Queries -getByPlaceholderText

Bu işlev, bir giriş alanının "placeholder" metnini kullanarak, bu alana kolayca erişmenizi sağlar. Placeholder metni, bir giriş alanının içine ne tür veriler girmeniz gerektiği hakkında ipuçları sunar ve genellikle gri bir renkle gösterilir.

İşte `getByPlaceholderText` işlemini kullanarak bir örnek:

```javascript
import { render, screen } from "@testing-library/react";

const TestComponent = () => (
	<div>
		<input type="text" placeholder="Search..." />
	</div>
);

it('should find the input field by its placeholder text', () => {
	render(<TestComponent />);
	const inputField = screen.getByPlaceholderText("Search...");
	expect(inputField).toBeInTheDocument();
});
```

Yukarıdaki örnekte, `getByPlaceholderText` işlevi "Search..." placeholder metnini kullanan giriş alanını seçer. Bu işlev, form alanları veya arama kutuları gibi, placeholder metni ile tanımlanan giriş öğelerini test etmek için çok kullanışlıdır.

Özetle, `getByPlaceholderText` işlevi, bir giriş alanını placeholder metni ile bulmak ve test etmek için kullanılır. Bu, form işlemleri ve kullanıcı girişi ile ilgili testler yaparken sık sık kullanılır.
