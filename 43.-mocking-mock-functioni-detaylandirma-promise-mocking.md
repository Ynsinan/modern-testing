# 🦥 43. Mocking - Mock Function'ı Detaylandırma - Promise Mocking

Bu bölümde bir önceki bölümde kullandığımız mock function'ı daha fazla nasıl detaylandırabiliriz bunların detayından bahsedeceğiz.&#x20;

Şimdi ilk örneklerimizi inceleyecek olursak her bir test case'ini yorum satırlarıyla inceleyerek ilerleceğiz İlk örneğimizle devam edelim.

{% code title="App.test.js" overflow="wrap" lineNumbers="true" %}
```javascript
import React from "react";

function TestComponent(){}

it("returns undefined by default", () =>{
 // Test edeceğimiz fonksiyonu jest.fn() ile mock oluşturuyoruz.
  const mock = jest.fn();

  // Mock fonksiyonu "foo" parametresi ile çağrıldığında, fonksiyonun dönüş değeri undefined olacak.
  let result = mock("foo");

  // Fonksiyonun dönüş değerini kontrol et, bu durumda beklenen değer undefined.
  expect(result).toBeUndefined();

  // Mock fonksiyonun çağrıldığını kontrol et.
  expect(mock).toHaveBeenCalled();

  // Mock fonksiyonun kaç kez çağrıldığını kontrol et, bu durumda beklenen değer 1.
  expect(mock).toHaveBeenCalledTimes(1);

  // Mock fonksiyonun belirli bir parametre ile çağrıldığını kontrol et, bu durumda beklenen değer "foo".
  expect(mock).toHaveBeenCalledWith("foo");
  }); 
```
{% endcode %}

{% code title="App.test.js" overflow="wrap" lineNumbers="true" %}
```javascript
it("mock implementation", () => {
  // Mock fonksiyonunu oluştururken, fonksiyonun yeni bir uygulamasını belirtiyoruz.
  const mock = jest.fn(() => "bar");

  // Mock fonksiyonunu "foo" parametresi ile çağırıyoruz ve dönüş değerini kontrol ediyoruz.
  expect(mock("foo")).toBe("bar");

  // Mock fonksiyonunun belirli bir parametre ile çağrıldığını kontrol ediyoruz.
  expect(mock).toHaveBeenCalledWith("foo");
});

```
{% endcode %}

{% code title="App.test.js" overflow="wrap" lineNumbers="true" %}
```javascript
it("mock implementation one time", () => {
  // Mock fonksiyonunu oluştururken, fonksiyonun uygulamasını belirliyoruz.
  const mock = jest.fn().mockImplementationOnce(() => "bar");

  // Mock fonksiyonunu "foo" parametresi ile ilk kez çağırıyoruz ve dönüş değerini kontrol ediyoruz.
  expect(mock("foo")).toBe("bar");

  // Mock fonksiyonunun belirli bir parametre ile çağrıldığını kontrol ediyoruz.
  expect(mock).toHaveBeenCalledWith("foo");

  // Mock fonksiyonunu ikinci kez çağırdığımızda, önceki uygulama belirlendiği gibi çalışmayacak.
  // Bu nedenle, ikinci çağrıda mock fonksiyonu önceki uygulama yerine varsayılan davranışını sergileyecek.
  expect(mock()).toBeUndefined();
});
```
{% endcode %}

`mockImplementationOnce` kullanarak bu fonksiyonun uygulamasını bir kez belirler. Bu durumda, fonksiyon bir kez çağrıldığında "bar" değerini dönecektir.&#x20;

{% code title="App.test.js" overflow="wrap" lineNumbers="true" %}
```javascript
it("mock return value", () => {
  // Mock fonksiyonunu oluşturuyoruz.
  const mock = jest.fn();

  // Mock fonksiyonunun dönüş değerini belirliyoruz.
  mock.mockReturnValue("bar");

  // Mock fonksiyonunu "foo" parametresi ile çağırıyoruz ve dönüş değerini kontrol ediyoruz.
  expect(mock("foo")).toBe("bar");

  // Mock fonksiyonunun belirli bir parametre ile çağrıldığını kontrol ediyoruz.
  expect(mock).toHaveBeenCalledWith("foo");
});

```
{% endcode %}

`mockReturnValue`Bu satır, oluşturulan mock fonksiyonunun dönüş değerini `"bar"` olarak belirler. Yani, bu mock fonksiyonu her çağrıldığında "bar" değerini dönecektir.

{% code title="App.test.js" overflow="wrap" lineNumbers="true" %}
```javascript
it("mock promise resolution", async () => {
  // Mock fonksiyonunu oluşturuyoruz.
  const mock = jest.fn();

  // Mock fonksiyonunun promise'ını oluşturup çözülme değerini belirliyoruz.
  mock.mockResolvedValue("bar");
  
  // Mock fonksiyonunu "foo" parametresi ile çağırıyoruz ve promise'ın çözülme değerini alıyoruz.
  const result = await mock("foo");

  // Promise'ın çözülme değerini kontrol ediyoruz.
  expect(result).toBe("bar");

  // Mock fonksiyonunun belirli bir parametre ile çağrıldığını kontrol ediyoruz.
  expect(mock).toHaveBeenCalledWith("foo");

  // Alternatif bir şekilde, mock fonksiyonunu çağırdığımızda direkt olarak resolve etmesini bekleyebiliriz.
  await expect(mock("foo")).resolves.toBe("bar");
});

```
{% endcode %}

{% code title="App.test.js" overflow="wrap" lineNumbers="true" %}
```javascript
it("mock promise rejection", async () => {
    // Hata nesnesini oluşturuyoruz.
    const error = new Error("Error message");

    // Mock fonksiyonunu oluşturuyoruz.
    const mock = jest.fn();

    // Mock fonksiyonunun promise'ını oluşturup reddetme değerini belirliyoruz.
    mock.mockRejectedValue(error);

    // İkinci bir mockRejectedValue çağrısı, ilk çağrıyı geçersiz kılar.
    mock.mockRejectedValue("return something");

    // mock fonksiyonunu "some arg" parametresi ile çağırdığımızda reject etmesini bekleriz
    await expect(mock("some arg")).rejects.toBe("return something");

    // mock fonksiyonunu "some arg" parametresi ile çağırdığımızda reject etmesini ve bir hata fırlatmasını bekleriz
    await expect(mock("some arg")).rejects.toThrowError();

    // mock fonksiyonunu "some arg" parametresi ile çağırdığımızda belirli bir hata nesnesi ile reject etmesini bekleriz
    await expect(mock("some arg")).rejects.toThrow(error);
});

```
{% endcode %}
