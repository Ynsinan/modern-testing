# 🖨️ 16. Queries - getByText

Bu işlev, bir sayfada belirli bir metni içeren öğeyi seçmek için kullanılır. Bu, belirli bir metni içeren bileşenleri veya öğeleri test etmek için sıkça kullanılır.

Aşağıda `getByText` işlevini kullanarak bir örnek gösterilmektedir:

```javascript
import { render, screen } from "@testing-library/react";

const TestComponent = () => (
	<div>
		<p>Hello, World!</p>
		<button>Click me</button>
	</div>
);

it('should find the text "Hello, World!"', () => {
	render(<TestComponent />);
	const textElement = screen.getByText("Hello, World!");
	expect(textElement).toBeInTheDocument();
});

it('should find the button by its text', () => {
	render(<TestComponent />);
	const button = screen.getByText("Click me");
	expect(button).toBeInTheDocument();
});
```

Yukarıdaki örnekte, `getByText` işlevi "Hello, World!" metnini içeren `<p>` öğesini ve "Click me" metnini içeren `<button>` öğesini bulur. Bu işlev, sayfanızdaki metinleri veya butonları test etmek için oldukça kullanışlıdır.

`getByText` işleminin bir diğer özelliği, metni bulduğunda bu metni içeren öğeyi döndürmesidir. Bu nedenle `getByText` kullanımının sonucunu bir değişkene atayabilir ve daha fazla işlem yapabilirsiniz.

Özetle, `getByText` işlevi, bir sayfada belirli bir metni içeren öğeyi seçmek ve test etmek için kullanılır. Bu işlev, kullanıcı arayüzü metinlerini ve öğelerini test etmek için sıkça kullanılır.
