# 🦝 46. Mocking - spyOn ile Fetch'i Mock'lamak

Bu adımda bir önceki konu başlığında `post-service.js` dosyamız içindeki axios requestleri fetch requeste çevirerek işe başlıyoruz.&#x20;

{% code title="post-service.js" lineNumbers="true" %}
```javascript
const getPosts = async () => {
    return fetch('https://jsonplaceholder.typicode.com/posts')
    .then((response) => response.json())
    .then((response) => response.data);
};

const getPost = async (id) => {
    return fetch(`https://jsonplaceholder.typicode.com/posts/${id}`)
   .then((response) => response.json())
    .then((response) => response.data);
}

const postService = {
    getPosts,
    getPost
};

export default postService;
```
{% endcode %}

Şimdi test tarafına tekrar dönelim.&#x20;

{% code title="App.test.js" lineNumbers="true" %}
```javascript
import { MOCK_POSTS } from "./mocks";
import postService from "./post-service";

// window.fetch fonksiyonunu jest.spyOn ile mockluyoruz.
const fetch = jest.spyOn(window, "fetch");

describe("<App />", () => {
  it("should be render bla bla", async () => {
    // window.fetch fonksiyonunu mocklamak için mockResolvedValueOnce kullanıyoruz.
    fetch.mockResolvedValueOnce({
      // Mock response'un içinde json fonksiyonunu tanımlayarak, json dönüşünü simüle ediyoruz.
      json: () => {
        // Json dönüşünü içeren bir Promise resolve ediyoruz.
        return Promise.resolve({
          data: MOCK_POSTS,
        });
      },
    });

    // postService.getPosts fonksiyonunu çağırıyoruz ve sonucu result değişkenine atıyoruz.
    const result = await postService.getPosts();

    // Sonucun, MOCK_POSTS dizisi uzunluğuna eşit olduğunu kontrol ediyoruz.
    expect(result).toHaveLength(MOCK_POSTS.length);

    // Sonucun, MOCK_POSTS dizisi ile aynı nesneleri içerdiğini kontrol ediyoruz.
    expect(result).toMatchObject(MOCK_POSTS);
  });
});

```
{% endcode %}

spyOn fonksiyonunundan bahsedecek olursak. `spyOn`, bir nesnenin belirli bir metodunu taklit ederek bu metodun çağrıldığını, kaç kez çağrıldığını ve çağrıldığı parametreleri izlememizi sağlayan bir Jest fonksiyonudur.

Nasıl Kullanılır?

```javascript
const myObject = {
  add: (a, b) => a + b,
};

// `add` metodunu spyOn ile taklit ediyoruz.
const spyOnAdd = jest.spyOn(myObject, "add");
```

```javascript
// Metodu çağırıyoruz
const result = myObject.add(2, 3);

// spyOn ile izlenen metodun çağrılma sayısı
console.log(spyOnAdd.mock.calls.length); // 1

// spyOn ile izlenen metodun çağrıldığı parametreler
console.log(spyOnAdd.mock.calls[0]); // [2, 3]

// spyOn ile izlenen metodun son çağrısının sonucu
console.log(spyOnAdd.mock.results[0].value); // 5

```

Basit şekilde anlatmaya çalıştım. İsterseniz kendiniz çeşitlendirerek deneyebilirsiniz.
