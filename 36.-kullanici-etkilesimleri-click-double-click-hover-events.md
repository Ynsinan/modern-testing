# 🐹 36. Kullanıcı Etkileşimleri - Click, Double Click, Hover Events

En çok karşımıza çıkması muhtemel olan şey tıklama olayı. Var olan [dökümantasyonda](https://testing-library.com/docs/user-event/convenience/) daha detaylı inceleyebilirsiniz. Direkt olarak kod üzerinden anlatmak daha sağlıklı olacak.&#x20;

#### Click Event (Tıklama Olayı)

{% code lineNumbers="true" %}
```javascript
import { render, screen } from '@testing-library/react';
import userEvent from '@testing-library/user-event';
import MyComponent from './MyComponent'; // Test edilecek bileşen

test('Click event works with userEvent', () => {
  const user = userEvent.setup();
  render(<MyComponent />);
  
  // Bileşen içindeki bir öğeyi seç
  const button = screen.getByRole('button');

  // Tıklama olayını taklit et
  user.click(button);

  // Beklediğiniz sonuçları kontrol et
  expect(/* Beklenen sonuçlar */).toBe(/* Beklenen değer */);
});

```
{% endcode %}

#### Double Click Event (Çift Tıklama Olayı)

{% code lineNumbers="true" %}
```javascript
import { render, screen } from '@testing-library/react';
import userEvent from '@testing-library/user-event';
import MyComponent from './MyComponent'; // Test edilecek bileşen

test('Double click event works with userEvent', () => {
  const user = userEvent.setup();
  render(<MyComponent />);
  
  // Bileşen içindeki bir öğeyi seç
  const element = screen.getByTestId('my-element');

  // Çift tıklama olayını taklit et
  user.dblClick(element);

  // Beklediğiniz sonuçları kontrol et
  expect(/* Beklenen sonuçlar */).toBe(/* Beklenen değer */);
});

```
{% endcode %}

#### Hover Event (Üzerine Gelme Olayı)

{% code lineNumbers="true" %}
```javascript
import { render, screen } from '@testing-library/react';
import userEvent from '@testing-library/user-event';
import MyComponent from './MyComponent'; // Test edilecek bileşen

test('Hover event works with userEvent', () => {
  const user = userEvent.setup();
  render(<MyComponent />);
  
  // Bileşen içindeki bir öğeyi seç
  const element = screen.getByTestId('my-element');

  // Mouse üzerine gelme olayını taklit et
  user.hover(element);

  // Beklediğiniz sonuçları kontrol et
  expect(/* Beklenen sonuçlar */).toBe(/* Beklenen değer */);

  // Mouse üzerinden ayrılma olayını taklit et
  userEvent.unhover(element);

  // Beklediğiniz sonuçları kontrol et
  expect(/* Beklenen sonuçlar */).toBe(/* Beklenen değer */);
});

```
{% endcode %}
