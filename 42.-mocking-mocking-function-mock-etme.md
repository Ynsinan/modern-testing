# 👾 42. Mocking - Mocking - Function Mock Etme

İlk başta tek bir tane mock function yaratmak üzerine ilerleyeceğiz. İlk olarak bir tane button componenti tasarlayalım.&#x20;

{% code title="button.jsx" lineNumbers="true" %}
```javascript
function Button({onClick, children}) {
   const handleOnClick = () => {
      onClick("Hey clicked");
   }

    return (
        <button onClick={handleOnClick}>{children}</button>
    )
}

export { Button };
```
{% endcode %}

Burada bir butonumuz var ve bir onClick fonksiyon alıyor. Buton tıklandığında çalışması için bir fonksiyon yazıyorum ve props olarak gelen onClick'i çalıştırıyoruz. Bu buton için çeşit çeşit onClick gelebilir. Spesifik olarak her fonksiyonu kontrol etmeyiz.&#x20;

{% code title="App.test.js" lineNumbers="true" %}
```javascript
import {render, screen} from '@testing-library/react';
import userEvent from '@testing-library/user-event';

import { Button } from './button';

function randomFunction(){};

describe('Button', () => {
  it("should call onClick Function when clicked", async () => {
    const user = userEvent.setup();
    render(<Button onClick={randomFunction}>Click me</Button>);
    const button = screen.getByRole('button', {name: /click me/i});
    await user.click(button);
    expect(randomFunction).toHaveBeenCalled();
  }
  )}
);
```
{% endcode %}

Burda random bir fonksiyon yazdık ne olduğu önemsiz. Daha sonrasında testimiz de Button componentimizi render ettik ve butona bir click olayı tetikledik. Daha sonrasında fonksiyonumuz çağırıldı mı diye kontrol ettik. Bu testi run ettiğimiz de hata alacağız. Çünkü her hangi bir random function verip tıklanıp tıklanmadığını kontrol etmek için bir fonksiyon veremeyiz. Bunu jest yardımıyla mocklamalıyız. Çünkü her fonksiyonu takip edemeyiz.&#x20;

{% code title="App.test.js" lineNumbers="true" %}
```javascript
import {render, screen} from '@testing-library/react';
import userEvent from '@testing-library/user-event';

import { Button } from './button';

//Burada fonksiyonumuzu jest yardımı ile mockluyoruz.
const randomFunction = jest.fn();

describe('Button', () => {
  it("should call onClick Function when clicked", async () => {
    const user = userEvent.setup();
    render(<Button onClick={randomFunction}>Click me</Button>);
    const button = screen.getByRole('button', {name: /click me/i});
    await user.click(button);

    expect(randomFunction).toHaveBeenCalled();
  }
  )}
);
```
{% endcode %}

Burada gelen fonksiyon ne olursa olsun fonksiyon çağırıldı mı kontrolümüzü yaparak. Fonksiyon mocklama işlemini öğrenmiş olduk.&#x20;
