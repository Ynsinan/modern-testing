# 🦦 45. Mocking - Npm Paketi Mock Etmek - Axios

Bu noktada axios paketini mock etmeyi öğreneceğiz.&#x20;

Basit bir şekilde 2 farklı request tanımlayarak başlayacağız. Basic bir service dosyası oluşturacağız ve içerisinde request modellerimizi yazacağız.

{% code title="post-service.js" lineNumbers="true" %}
```javascript
import axios from 'axios';

const getPosts = async () => {
    return axios.get('https://jsonplaceholder.typicode.com/posts')
    .then((response) => response.data);
};

const getPost = async (id) => {
    return axios.get(`https://jsonplaceholder.typicode.com/posts/${id}`)
    .then((response) => response.data);
}

const postService = {
    getPosts,
    getPost
};

export default postService;
```
{% endcode %}

Şimdi app.test.js dosyamıza dönelim.&#x20;

{% code title="App.test.js" lineNumbers="true" %}
```javascript
import { MOCK_POSTS } from "./mocks";
import postService from "./post-service";
import axios from "axios";

// axios modülünü mocklamak için jest.mock kullanılır.
jest.mock("axios");

describe("<App />", () => {
  it("should be render bla bla", async () => {
    // Mock bir axios response'u oluşturuyoruz.
    const mockResponse = { data: MOCK_POSTS };

    // axios.get fonksiyonunu mocklayarak, bu fonksiyonun belirli bir değeri resolved etmesini sağlıyoruz.
    axios.get.mockResolvedValueOnce(mockResponse);

    // postService.getPosts fonksiyonunu çağırıyoruz ve sonucu bir değişkene atıyoruz.
    const result = await postService.getPosts();

    // Sonucu console.log ile yazdırıyoruz. Bu satır test sırasında geliştiriciye yardımcı olabilir.
    console.log(result);
  });
});
```
{% endcode %}

Bu noktada testimizi run ettiğimiz de hata alabiliriz. Sebebi axios kaynaklı terminalde Cannot use import statement outside a module şeklinde bir error görebilirsiniz. Bunun çözümü için şöyle bir yöntem uygulayabilirsiniz.&#x20;

```javascript
"jest": { "transformIgnorePatterns": [ "node_modules/(?!axios)" ] }
```

package.json dosyamızın içine bunu ekleyerek o hatadan kurtulabiliriz.&#x20;

Bu noktada testimizi run ettiğimiz de terminalimiz de result logunu görebiliriz.  Son olarak bi assertion işlemi gerçekleştirelim.

{% code title="App.test.js" lineNumbers="true" %}
```javascript
describe("<App />", () => {
  it("should be render bla bla", async () => {
    const mockResponse = { data: MOCK_POSTS };
   axios.get.mockResolvedValueOnce(mockResponse);
   const result = await postService.getPosts();
   
   expect(result).toHaveLength(MOCK_POSTS.length);
   expect(result).toMatchObject(MOCK_POSTS);
  })
});
```
{% endcode %}

Ek olarak catch bloğu koyarak bir test yazmaya çalışalım. Burada `getPosts` fonksiyonumuza bir catch bloğu ekleyip bir error message return ediyoruz.&#x20;

{% code title="post-service.js" lineNumbers="true" %}
```javascript
const getPosts = async () => {
    return axios.get('https://jsonplaceholder.typicode.com/posts')
    .then((response) => response.data).catch((error) => {
        return {error: {
            message:"Data Bulunamadı",
        }}
    });
};
```
{% endcode %}

Şimdi testini yazmaya çalışalım.&#x20;

{% code title="App.test.js" lineNumbers="true" %}
```javascript
it("should be reject axios", async () => {
    const mockResponse = { data: MOCK_POSTS };
    axios.get.mockRejectedValue(mockResponse);
    const result = await postService.getPosts();
    
    expect(result).toMatchObject({error: {
      message:"Data Bulunamadı",
    }});
   }
  )
```
{% endcode %}

Terminalde bu şekilde success olduğunu görebiliriz. Siz de isterseniz kendiniz farklı assertionlar deneyebilirsiniz.&#x20;
