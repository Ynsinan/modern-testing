# 🕵🏽‍♂️ 19. Queries - getByTitle

Bu işlev, bir HTML öğesinin `title` özniteliği (attribute) ile eşleşen bir başlık (title) kullanarak bu öğeyi seçmenizi sağlar. `title` özniteliği, bir öğenin üzerine geldiğinizde veya üzerine odaklandığınızda görünen metni temsil eder.

`getByTitle` işlemini kullanarak bir örneği inceleyelim:

```javascript
import { render, screen } from "@testing-library/react";

const TestComponent = () => (
	<div>
		<a href="/about" title="About Us">Learn more about us</a>
		<button title="Click me">Click me</button>
	</div>
);

it('should find the link by its title', () => {
	render(<TestComponent />);
	const link = screen.getByTitle("About Us");
	expect(link).toBeInTheDocument();
});

it('should find the button by its title', () => {
	render(<TestComponent />);
	const button = screen.getByTitle("Click me");
	expect(button).toBeInTheDocument();
});

```

Yukarıdaki örnekte, `getByTitle` işlevi "About Us" başlığına sahip olan `<a>` bağlantısını ve "Click me" başlığına sahip olan `<button>` öğesini seçer. Bu işlev, sayfanızdaki öğelerin üzerine gelindiğinde veya üzerine odaklandığında görünen başlıkları doğrulamak ve test etmek için kullanışlıdır.

`getByTitle` işlevi, özellikle kullanıcıların belirli bir öğenin üzerine gelerek daha fazla bilgi edinmeleri gerektiği durumlarda kullanışlıdır. Bu işlev, kullanıcı deneyimi ve erişilebilirlik testleri yaparken yardımcı olabilir.

Özetle, `getByTitle` işlevi, bir öğenin `title` özniteliği ile eşleşen başlığı kullanarak bu öğeyi seçmenizi ve test etmenizi sağlar. Bu işlev, özellikle kullanıcı arayüzünde başlıkları doğrulamak için kullanılır.
